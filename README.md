The following GitHub repository contains Appendix files for the publication:

Simulating telemetry studies that estimate component mortality rates of imperiled juvenile salmonids

Authors: E. M. Greenheck, M. Peterson, T.J. Pilger, M. A. Djokic, J. Eschenroeder, T. R. Nelson

Each .R file contains simulation code for three- (Appendices S1 and S2) and four- (Appendix S3) state multistate mark-recapture models written in Bayes. The .pdf files describe each .R file in greater detail.
